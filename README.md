# AI Interview Simulator

AI Interview Simulator is a web-based application that simulates real technical and HR interviews using AI-powered chat interaction. It helps candidates practice interview questions, receive instant feedback, and improve confidence before real interviews.

## Features

- Simulates technical and HR interview questions
- Asks one question at a time like a real interview
- Provides instant AI feedback and improvement suggestions
- Clean and responsive user interface
- Helps users evaluate answers and communication skills

## Tech Stack

- Next.js (App Router)
- TypeScript
- AI SDK / OpenAI API
- Tailwind CSS

## How It Works

1. The AI acts as an interviewer.
2. Users respond to interview questions via chat.
3. After each response, the AI provides feedback and improvement tips.
4. At the end, the user receives an overall interview evaluation.

## Use Case

Many candidates lack access to realistic interview practice. This project provides a simulated interview environment to help users prepare effectively and build confidence.
